vnc2rdp
=======

**vnc2rdp** is a proxy for RDP client connect to VNC server, released under the [**Apache License, Version 2.0**](http://www.apache.org/licenses/LICENSE-2.0).

Currently, vnc2rdp support:

RDP client:

* mstsc.exe
* [freerdp](http://www.freerdp.com/)

VNC server:

* RealVNC (depth 24)
